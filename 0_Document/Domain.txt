===기본 규칙===
컬랙션은 편의상 []로 표기하지만 실제로는 불변이어야 함
지역 변수 : _camelCase
매개변수 등 지역 변수 제외 모든 변수 : camelCase
함수, 프로퍼티, 클래스, 구조체, 레코드, 폴더, 파일명 : PascalCase

===Start Game===
Unity Start -> StageIndex LoadStageProgress(string progressKey) -> StageModel[] CreateStageModels(StageProgress progress) -> void DrawStageUI()

-LoadStageProgress : PlayerPrefabs 사용
-CreateStageModels : 진행도보다 큰 스테이지는 isClear false, 아니면 true


===Puzzle Play===
-ChessSquare SelectSquare() -> SelectedPiece GetSquarePieces(ChessSquare square) -> PuzzlePlayState CreateMoveState(SelectedPiece) -> State Change And Event
-

===Types===
PiecesType : King | Queen | Rook | Bishop | Knight | Pawn | None
MoveRangeTpye : Infinite, Range, Fix,
MoveType : Jump, Slide
PuzzleState : Select | Move | Clear | Fail

===Records===
StageIndex(int chapter, int stage) : 비교 재정의, 챕터가 큰 쪽이 큼, 같다면 스테이지가 큰 쪽이 큼, 둘이 같으면 동등
StageModel(int chapter, int stage, bool isClear)

ChessSquare(string file, int rank) : file은 1~8, rank는 1~8
SelectedPiece : Option<PiecesType> (Some | None)
PuzzlePlayState(PuzzleState puzzleState, SelectedPiece pieces)


===Move===
And는 하나의 규칙으로 동시에 적용됨, Or는 별도의 규칙으로 이동 가능 영역만 늘어남
file, rank, MoveType
King : (+-1 Or 0) And (+-1 Or 0), Jump
Quenn : Rook Or Bishop, Slide
Rook : (+-N And 0) Or (0 And +-N), Slide (N은 무한대)
Bishop : +-N And +-N (N은 무한이며 0이 아니고 무조건 같은 값), Slide
Knight : (+-2 And +-1) Or (+-1 And +-2), Jump
Pawn : +1 And +-1 And, Jump
